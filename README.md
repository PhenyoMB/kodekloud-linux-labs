# KodeKloud Linux Labs

This repository contains solutions, commands and key takeaways from hands-on Linux labs completed on KodeKloud.

## Purpose

This repository is used to document practical Linux administration tasks in a concise and professional format. Each lab includes the task completed, the commands used, what was learned, and short personal reflections.

## Repository Structure

Each lab is stored in its own folder and documented with a dedicated `README.md`.

```text
kodekloud-linux-labs/
├── README.md
└── linux-level-01
    └── day-01-custom-apache-user-setup/
        └──README.md
```
